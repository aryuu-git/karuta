## 📖 游戏概述

歌牌大乱斗是一款**多人实时抢牌游戏**。房主选择一个牌组，系统依次播放牌组里每首歌曲的音频，玩家需要快速找到并点击棋盘上对应的那张牌。
听音辨牌，手速决定胜负！(ง •̀_•́)ง

## 🃏 牌组

### 什么是牌组？
牌组是游戏的核心素材库，由玩家自己创建和管理。每个牌组包含若干张牌，每张牌包含：

| 内容 | 说明 |
|------|------|
| **封面图** | 牌的正面图片（必须上传，游戏中展示） |
| **音频文件** | 对应这张牌的歌曲（必须上传，游戏中播放） |
| **歌曲名称** | 选填，仅在牌组管理页显示，游戏中不可见 |
| **播放提示** | 选填，音频播放时显示在读牌区（上句提示） |

### 支持的格式
- **音频**：mp3、wav、m4a、flac、ogg、aac（单文件 ≤ 20MB）
- **封面图**：jpg、png、webp（单文件 ≤ 5MB）

### 牌组共享
- 默认牌组为**私有**，只有自己能用
- 可以将牌组设为**公共**，其他玩家可在首页"公共牌组库"看到并直接用来开房间
- 设为公共后随时可以改回私有

## 🏯 房间与游戏模式

### 创建房间
1. 进入首页，点击"创建房间"
2. 选择一个牌组（必须至少有1张牌）
3. 设置**朗读间隔**（3 ~ 30秒）：音频播放完毕后，留给玩家抢牌的缓冲时间
4. 选择游戏模式
5. 创建后获得6位**邀请码**，分享给朋友

### 加入房间
- 在首页"战场大厅"找到招募中的房间，直接点击加入
- 或输入邀请码加入
- **激战中的房间不可加入**（只有招募中的房间开放加入）

## 🎮 游戏模式

### 模式一：🤖 自动模式

**适合多人在线一起玩**

1. 所有人加入房间后，房主点击**「全员集合！开战！」**
2. 游戏开始前有 **3-2-1 倒计时**
3. 系统从牌组中**随机选取**一首歌开始播放
4. 棋盘上展示本局所有牌的封面，**不显示任何文字**（需要靠听歌识牌）
5. 音频播放完毕后进入**缓冲期**（即设定的朗读间隔时长），缓冲期内仍可抢牌
6. 缓冲期结束进入下一首，循环直到所有牌都结束
7. 所有牌结束后自动进入结算页

### 模式二：👑 裁判模式

**适合线下聚会，由一人主持游戏**

1. 房主作为**裁判**，其他玩家参与抢牌
2. 裁判看到的是全部牌的列表，可以**任意选择播放哪首**
3. 裁判点击一张牌 → 音频广播给所有玩家 → 玩家抢牌
4. 一首结束后，界面提示裁判选择下一首
5. 裁判不参与抢牌、不计分

## ⚡ 抢牌规则

### 基本流程
1. 音频开始播放
2. **任意时刻**均可点击棋盘上的牌（整个播放过程中都可以抢）
3. 音频结束后进入缓冲期，缓冲期内仍然可以抢牌
4. 缓冲期结束，本首结束，进入下一首

### 判定规则
服务端以**消息到达时间**为权威判定依据，完全防止客户端作弊。

| 结果 | 条件 | 得分变化 |
|------|------|---------|
| ✅ 抢牌成功 | 第一个点击当前播放的牌 | **+1 分** |
| ❌ 抢错（已被抢） | 点击了当前牌，但已被别人先抢 | **-1 分**，本首禁止继续抢牌 |
| ❌ 抢错（错误的牌） | 点击了不是当前播放的牌 | **-1 分**，本首禁止继续抢牌 |
| 🚫 本首出局 | 本首已抢错过一次 | 点击无效，只能观望 |

> **注意**：分数可以为负数！抢错次数多会导致负分。

### 全员出局
- 若本首所有在线玩家都已抢错，游戏自动结束本首，进入下一首
- 提示：「💀 全员出局！本首自动结束」

### 无人抢牌
- 若音频播完 + 缓冲期结束均无人抢牌，本首自动结束
- 该牌不计入任何人得分

## 📊 计分规则

| 情况 | 分值 |
|------|------|
| 成功抢牌 | +1 |
| 抢错（任意原因） | -1 |
| 无人抢 | 0 |

- 分数**可以为负数**
- 每首歌开始时，本首的禁止状态**重置**（下一首所有人都可以重新抢）
- `wrongUsers`（本首出局）在每首新牌开始时清空

---

## 🎯 棋盘说明

- 棋盘展示本局所有牌的**封面图**
- **不显示文字**（歌名、歌词等），需要靠听歌识牌
- 牌的显示顺序在开局时**随机打乱**，且每个玩家的顺序相同（服务端统一分配）
- 被抢走的牌缩小显示在棋盘底部，显示抢牌者名字
- 正在播放的牌**不会高亮**（防止作弊）

---

## ⏸ 暂停与继续

- 仅**房主**可以暂停和继续游戏（自动模式）
- 暂停时音频暂停，计时器冻结，玩家无法抢牌
- 继续后音频恢复，计时器从暂停处继续倒计时

---

## 🏆 结算

所有牌打完后进入结算页：
- 按得分**从高到低**排列
- 显示 🥇🥈🥉 前三名特效
- 第一名有樱花粒子特效
- 可选择「再战一次」或「回家休息」

## 👑 特殊权限

### 房主权限
- 开始游戏（waiting 状态）
- 暂停/继续游戏
- 解散战场（强制结束房间）

- 可在**游戏内**通过控制栏强制结束当前对局

## ❓ 常见问题

**Q：音频播放很长，什么时候可以抢？**
A：整个音频播放过程中随时可以抢，不受间隔时间限制。间隔时间是音频播完后的缓冲等待期。

**Q：点了一张牌没有反应？**
A：可能是当前没有牌在播放（等待状态），或你本首已经抢错被禁止。

**Q：我的分数变成负数了？**
A：抢错次数过多会扣分，分数可以为负。

**Q：为什么棋盘上的牌没有文字？**
A：故意设计的！需要通过听音频来判断是哪张牌，这才是游戏的核心玩法。

**Q：我怎么知道当前播放的是哪首？**
A：如果上传牌时填写了「播放提示」，读牌区会显示提示文字。否则只能靠听力判断！

**Q：可以中途加入游戏吗？**
A：不可以。激战中的房间禁止新玩家加入，只有「招募中」状态的房间才能加入。
